# Landing Page Política - Laura Jiménez

Landing page desarrollada como proyecto académico para el programa de Análisis y
Desarrollo de Software (ADSO) - SENA CTMA. Presenta la propuesta de campaña de una
candidata ficticia, con fines exclusivamente educativos.

## Estructura del proyecto

## Tecnologías utilizadas

- HTML5
- CSS3 (diseño responsive con Flexbox y Grid)
- JavaScript (Vanilla JS)

## Secciones de la página

- **Inicio:** presentación principal de la candidata.
- **Propuestas:** principales ejes de la campaña (educación, empleo, seguridad, medio ambiente).
- **Biografía:** información sobre la candidata.
- **Contacto:** formulario de contacto con validación básica.

## Cómo verlo localmente

Solo abre el archivo `index.html` en tu navegador, o usa una extensión como
"Live Server" en VS Code.
